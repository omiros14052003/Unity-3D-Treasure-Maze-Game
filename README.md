# Unity-3D-Treasure-Maze-Game
A simple 3D treasure maze game made in Unity for the course MYY702 Computer Graphics of CSE UOI.

## Notes for explanation of the project
The documentation of the project is given with the .pdf file. The actual project files cannot be uploaded to GitHub due to the size. The zip of the project is given through the following google drive link: https://drive.google.com/file/d/179cQ-M6fwubeqQK663Ekzr8t-CXXdSt3/view?usp=sharing
The link in the .pdf was for the project's examination by the faculty and it is connected to the personal academic account, so it might be unavailable.

## Running the app
The project was set up for the Unity Editor version 2020.3.24f1.
